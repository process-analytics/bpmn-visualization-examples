BPMN Visualization Examples

Directly access to bpmn-visualization live examples. Technical details about the implementation are available in the bpmn-visualization-examples repository.

Extensibility

  • custom colors: change default fill and stroke colors, font color, use specific colors depending on the BPMN element
  • custom fonts: change default font, font color, use specific font depending on the BPMN element
  • custom user task icon: use your own icon
